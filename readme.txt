README pour le premier blog en php

installation du projet : 

- il faut avoir un moyen de lancer un serveur local de développement (wamp64 par exemple)
- dans le répertoire "www" pour wamp64 ou bien dans le répertoire correspondant de votre serveur 
    copier le projet avec git => "git clone https://github.com/AurelienCorp/blogPHP.git"
- créer une base de données avec le fichier sql fournis dans le dossier SQL_Database
- mettre à jours les données présentent dans le .env avec les données que vous avez en local pour la base de données
- avec le serveur lancer le localhost correspondant au projet

Fonctionnement du site :

- via la page d'accueil vous pouvez consulter les 3 articles les plus récents publiés sur le site
- vous pouvez télécharger le CV au format PDF
- accès au réseaux sociaux
- via la barre de navigation vous avez accès à la liste de tout les articles ou bien des articles par catégories
- vous pouvez vous inscrire sur le site en tant qu'abonné en remplissant le formulaire qui se trouve en bas de page
    cela vous donne le droit des laisser des commentaires sur les articles
- la connection à votre compte se trouve dans la barre de navigation en haut a droite 

Administration : 

- si vous avez les droits administrateur vous pouvez accéder à la partie administration en cliquant sur le lien dans le pied de page
- sur la page d'accueil de l'administation vous avez accès à la gestion des commentaires ainsi qu'à la création et modification des articles
- vous pouvez revenir sur le site en cliquant sur le lien en bas de page
